NXZ Design System

Sistema de design para a plataforma NXZ — SaaS de gestao de franquias de food service. Inspirado no minimalismo elegante do Square com tipografia serifada, CTAs em pill shape, cinzas quentes e espacamento generoso ao estilo Google Antigravity.

Plataforma

Web, Mobile-first

Tema

Light only (sem dark mode)

Inspiracao

Square + Google Antigravity

Prefixo CSS

--nxz-*

Principios de Design

Clareza

Hierarquia visual clara com tipografia serifada para headlines e sans-serif para corpo. Cada elemento tem um proposito definido.

Espacamento Generoso

Whitespace abundante entre secoes e componentes. O conteudo respira, criando uma experiencia premium e legivel.

Consistencia

Tokens centralizados garantem uniformidade visual em toda a plataforma. Pill shapes nos CTAs, cantos arredondados suaves nos cards.

Cores

Paleta de cores quentes com roxo vibrante como cor primaria. Clique em qualquer swatch para copiar o valor HSL.

Acessibilidade

Diretrizes para garantir que a plataforma NXZ seja acessivel a todos os usuarios, seguindo WCAG 2.1 nivel AA.

Contraste de Cores

Texto principal (preto sobre off-white) atinge ratio 19:1. Texto muted (44% cinza) atinge 4.7:1, acima do minimo AA de 4.5:1. Botoes primary (branco sobre roxo) atingem 5.2:1.

Focus Visible

Todos os elementos interativos possuem :focus-visible com ring roxo de 2px e offset de 2px. Nunca remova o outline sem fornecer alternativa visivel.

Tamanho de Toque

Botoes e inputs tem altura minima de 48px, atendendo ao requisito de area de toque de 44x44px do WCAG. Espacamento entre alvos de toque: minimo 8px.

Hierarquia de Headings

Use headings em ordem sequencial (h1 > h2 > h3). Apenas um h1 por pagina. Nao pule niveis. Headings em DM Serif facilitam o escaneamento visual.

ARIA Labels

Icones decorativos: aria-hidden="true". Icones funcionais: aria-label="descricao". Botoes com so icone devem ter aria-label.

Navegacao por Teclado

Todos os componentes devem ser operaveis via teclado. Tab para navegar, Enter/Space para ativar, Escape para fechar. Modais fazem trap de foco.

Reduced Motion

Respeitar prefers-reduced-motion. Quando ativa, remover ou reduzir todas as animacoes e transicoes para usuarios sensiveis a movimento.

Texto Alternativo

Todas as imagens significativas devem ter alt descritivo. Imagens decorativas: alt="". Graficos devem ter descricao textual alternativa.

/* Reduced motion */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}

.nxz-btn:focus-visible,
.nxz-input:focus-visible {

  outline-offset: 2px;
}

Densidade

O NXZ usa density mode "comfortable" — alturas generosas para toque mobile-first e espacamento amplo para legibilidade.
